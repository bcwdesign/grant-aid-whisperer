# GrantPilot

GrantPilot is an AI-powered SaaS platform that helps nonprofits discover, track, and plan grant applications.

## Core Features

- AI grant discovery
- Smart grant matching
- Deadline tracking
- Grant planning workspace
- Task management

---

## Tech Stack

Frontend:
- Next.js
- Tailwind CSS

Backend:
- Node.js / Supabase
- PostgreSQL

AI:
- TinyFish grant agents
- OpenAI

---
